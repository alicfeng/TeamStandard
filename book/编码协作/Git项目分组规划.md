# Git项目分组规划

|     命名      |    描述    |            对象            |
| :-----------: | :--------: | :------------------------: |
| `application` |   应用层   | 网格服务、前后端等代码仓库 |
|  `framework`  |  算法框架  |      算法基础环境框架      |
|     `sdk`     |  标准SDK   |       所有底层 `SDK`       |
| `dockerfile`  | dockerfile |     所有 `dockerfile`      |
|  `document`   |    文档    | 所有需要 `git` 整合的文档  |
|    `tool`     |    工具    |       开发等所有工具       |
